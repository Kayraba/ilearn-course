Save a file
===========

01. Lesson 6: Save a file. I will guide you through it one step at a time.

02. In this lesson, we save work so we can find it again later. I will show you how to write a short note, give it a clear file name, save it in Documents, close it, and open it again.

03. We have written a short note. To keep it, we click the File menu at the top, and then choose Save.

04. The computer asks for a name. We give it a simple, clear name, so we can find it again later.

05. And we always save to the same place: the Documents folder. Then we press Save.

06. Later, we open Documents, and our work is waiting for us. The same drawer, every time.

07. When you start the activities, they will follow the same order as this lesson. First, you do a warm-up: Pick a meaningful name such as shopping list, appointment note or my practice note. Then you answer quick checks, so we know the key words make sense. Next, you practise the real computer skill: Open Word, type one sentence, use Save As, type the file name and choose Documents. You may also type a short answer, for example: Type a clear file name. At the end, you finish with a stretch or reflection: Use Save As to make a second copy with a different name. Talk about when two versions might be useful. Take one activity at a time. If you need help, ask staff, then try the next step yourself.

08. Here is one useful thing to watch for. Closing without saving can lose new work. Always save first, then close. If it happens, pause, reset, and try again calmly.

09. Wonderful. You can save your work with a clear name in Documents — and that means it will never be lost. Now it is your turn. Scroll down to the activities and do one step at a time. Ask staff for help if you need it — that is part of learning.
