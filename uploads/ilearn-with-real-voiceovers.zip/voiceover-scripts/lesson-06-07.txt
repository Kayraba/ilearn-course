When you start the activities, they will follow the same order as this lesson. First, you do a warm-up: Pick a meaningful name such as shopping list, appointment note or my practice note. Then you answer quick checks, so we know the key words make sense. Next, you practise the real computer skill: Open Word, type one sentence, use Save As, type the file name and choose Documents. You may also type a short answer, for example: Type a clear file name. At the end, you finish with a stretch or reflection: Use Save As to make a second copy with a different name. Talk about when two versions might be useful. Take one activity at a time. If you need help, ask staff, then try the next step yourself.
