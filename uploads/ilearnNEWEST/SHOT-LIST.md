# Confident Computing — Shot List (screens to capture for each video)

What to grab for every lesson. **Most are still screenshots** (easiest). A few need a
**short 3–8 second clip** because the movement is the point — those are marked 🎬.
Physical things (power button, keyboard) you can just photograph with your phone 📷.

**How to capture on Windows:** `Windows + Shift + S` = screenshot a region · `Windows + G`
or Clipchamp = short screen clip. Turn the **mouse pointer large + bold** first
(Settings → Accessibility → Mouse pointer). Capture at full screen for crisp video.

## If you'd rather generate them (AI image prompt)
For any screen you want generated instead of captured, use a prompt like:

> "A clean, simplified screenshot of [Microsoft Word / Excel / the Windows desktop]
> showing [exactly what — e.g. the word 'David' selected in blue and made bold, with the
> Bold button highlighted]. Large, clear, high-contrast, minimal clutter, 1920×1080,
> flat UI style, soft neutral background. No real logos."

But honestly, real captures from your own PC beat generated ones for teaching — generated
UIs invent buttons that aren't really there. Capture the real ones where you can.
