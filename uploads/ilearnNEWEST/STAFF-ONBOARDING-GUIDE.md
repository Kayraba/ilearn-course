# Staff Onboarding Guide

## What iLearn does

iLearn helps supported learners practise computer basics, Word and Excel through
short demonstrations, guided practice cards and staff-recorded progress.

## Starting a learner

1. Ask the learner to enter their name on the start screen.
2. Use the same name each session so progress continues.
3. If two learners share a name, agree a safe consistent label with staff, such
   as first name plus initial.

## Running a lesson

1. Check comfort, access needs and screen visibility.
2. Watch or present the lesson demonstration.
3. Work through the guided practice cards.
4. Give one prompt at a time and wait.
5. Record the support level when marking the lesson complete.

## Prompt levels

- Independent: no help.
- Verbal: spoken reminder.
- Gesture: pointing/visual cue.
- Physical: hand-over-hand or direct physical support.

## Admin tasks

- add/manage learners
- view progress
- record consent/capacity
- export CSV reports
- view activity log
- remove learner records if required by the pilot process

## Good practice

- keep notes factual and respectful
- avoid unnecessary sensitive information
- pause if the learner is tired or distressed
- celebrate small progress
