# Accessibility Statement - iLearn Controlled Pilot

Status: pilot statement. iLearn has not yet had an external WCAG audit.

## Accessibility aims

iLearn is designed for supported adults, including learners with learning
disabilities. It aims to provide:

- clear language
- large touch targets
- visible focus states
- simple navigation
- labelled form fields
- adjustable text size
- calm animations
- reduced-motion support where possible
- captions inside lesson animations

## Current known limitations

- The animated walkthroughs are visual and may need staff narration or captions
  in video production for some learners.
- Name-based learner continuation is accessible but not a strong identity check.
- Formal WCAG 2.2 AA testing has not yet been completed.
- Some lesson tasks are staff-observed real-computer activities, so completion
  quality depends on staff support.
